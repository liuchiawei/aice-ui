# Aice UI

Next.js、React、Tailwind CSS、shadcn/ui スタイルで構築した UI コンポーネント。プロジェクトにコピーするか CLI で追加できます。

**プレビュー:** [aice-ui.vercel.app](https://aice-ui.vercel.app/)

---

## インストールと利用

### 必要環境

- Node.js 18+
- pnpm 推奨

### ローカルで起動

```bash
git clone https://github.com/your-username/aice-ui.git
cd aice-ui
pnpm install
pnpm dev
```

[http://localhost:3000](http://localhost:3000) を開きます。

### 自分のプロジェクトで使う

- **コピー** — `components/ui/`、`components/background/` などから `.tsx` をアプリにコピーする。
- **shadcn CLI** — プロジェクトの参照先をこのリポジトリの registry にして実行：

  ```bash
  pnpm run build:registry   # このリポジトリで先に実行
  pnpm dlx shadcn@latest add <component>
  ```

### CLI（グローバル）

パッケージ公開後：

```bash
pnpm add -g @doublecheap/aice-ui
```

CLI から registry 経由でコンポーネントを追加できます。

## コンポーネント一覧

カテゴリ別（`lib/component-config.ts` を参照）。

### UI

| コンポーネント | 説明 |
|----------------|------|
| **リキッド・グラス** | ぼかし付きグラス面。 |
| **コードブロック** | シンタックスハイライトと行番号付きコードブロック。 |
| **コマンドライン** | 一行コード・コマンド表示とコピーボタン。複合: Provider、Addon、CopyButton。 |
| **ポップアップアイテム** | ポップアップアニメーションで内容を表示するコンテナ（表示時・ホバー・クリックで発火）。 |

### テキスト

| コンポーネント | 説明 |
|----------------|------|
| **グリフ片仮名** | 解碼アニメーション付きテキスト。片仮名が目標テキストへ。 |
| **解碼テキスト** | ホバーまたは表示時に解読されるスクランブルテキスト。 |
| **テキストホバー効果** | ホバー時カーソル追従のグラデーションとストローク。 |
| **スライドアップレター** | オプションのタイトル・説明付きスライドアップ文字。 |
| **アニメーションカウンター** | 表示時に開始値から目標値へアニメーションする数字。 |
| **タイマー** | 経過時間タイマー。開始・一時停止・リセット。表示形式を指定可能。 |

### ボタン

| コンポーネント | 説明 |
|----------------|------|
| **コピーボタン** | テキストをクリップボードにコピー。コピー後にチェック表示。 |
| **スクロールボタン** | ウィンドウをスクロール（上・下・左・右）。 |
| **テーマ切り替え** | ダーク／ライト切り替え（React、Tailwind、Motion）。 |

### カード

| コンポーネント | 説明 |
|----------------|------|
| **3Dカード** | ホバーで3D傾斜。パララックス深度。 |
| **3Dフリップカード** | 3D傾斜とクリックで裏表が入れ替わるフリップカード。 |

### チャート

| コンポーネント | 説明 |
|----------------|------|
| **ラダーチャート** | ラダーチャート。オプションでタイトル・説明。 |
| **放射状クラスター図** | 階層データのラジアルクラスター図。data prop または Item/SubItem 構成に対応。 |

### カルーセル

| コンポーネント | 説明 |
|----------------|------|
| **回転カルーセル** | 回転カルーセル。前後・ドット・オプションの中央パネル。 |

### レイアウト

| コンポーネント | 説明 |
|----------------|------|
| **ドラッググリッド** | ドラッグ可能な2Dグリッド。セルにパララックス。 |
| **タイムライン** | 縦型タイムライン。スティッキーな年とカード内容。 |

### ナビゲーション

| コンポーネント | 説明 |
|----------------|------|
| **スピードダイヤル** | FAB が放射状メニューに展開（Root、Trigger、Item）。 |
| **フローティングドック** | 下部ドック。ホバーで拡大。モバイルはメニューに折りたたみ。 |

### 背景

| コンポーネント | 説明 |
|----------------|------|
| **浮動サークル** | 浮動円形背景。オプションでタイトル・説明。 |
| **マトリックスコード** | マトリックス風コード。グリッチとオプションのビネット。 |
| **波状グリッド** | Canvas パーリンノイズ波状グリッド。マウス・タッチ操作。 |
| **星空背景** | Canvas 星空。星の数・きらめきを設定可能。 |

### メディア

| コンポーネント | 説明 |
|----------------|------|
| **アスキーアート** | 画像を ASCII アートに。オプションでフェード・タイプライター・マトリックス。 |
| **動くピクセル** | 画像をドットグリッドに。オプションでポインター変形。 |
| **ヒートマップキャンバス** | 画像にヒートマップグラデーション（thermal、viridis、jet など）。 |
| **ウェブカメラ** | ウェブカメラ。オプションでタイトル・説明。 |

**合計 30 コンポーネント**
