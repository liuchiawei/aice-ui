# OPEN-UI

以 Next.js、React、Tailwind CSS 與 shadcn/ui 風格建構的 UI 元件。可複製到專案或透過 CLI 加入。

**線上預覽：** [aice-ui.vercel.app](https://aice-ui.vercel.app/)

---

## 安裝與使用

### 需求

- Node.js 18+
- 建議使用 pnpm

### 本地執行

```bash
git clone https://github.com/your-username/open-ui.git
cd open-ui
pnpm install
pnpm dev
```

開啟 [http://localhost:3000](http://localhost:3000)。

### 在專案中使用

- **複製** — 從 `components/ui/`、`components/background/` 等複製 `.tsx` 到你的應用。
- **shadcn CLI** — 將專案指向本 repo 的 registry 後執行：

  ```bash
  pnpm run build:registry   # 請先在本 repo 執行
  pnpm dlx shadcn@latest add <component>
  ```

### CLI（全域）

套件發佈後：

```bash
pnpm add -g @doublecheap/open-ui
```

之後可透過 CLI 從 registry 加入元件。

## 元件一覽

依類別分組（見 `lib/component-config.ts`）。

### UI

| 元件 | 說明 |
|------|------|
| **液態玻璃** | 具模糊與變形效果的玻璃表面。 |
| **代碼** | 具語法高亮與行號的程式碼區塊。 |
| **指令欄** | 單行程式碼或指令顯示，附複製按鈕。複合：Provider、Addon、CopyButton。 |
| **彈出項目** | 以彈出動畫顯示內容的容器（進入視窗、懸停或點擊觸發）。 |

### 文字

| 元件 | 說明 |
|------|------|
| **片假名符號** | 解碼動畫文字，片假名逐漸揭示目標文字。 |
| **解碼文字** | 懸停或進入視窗時解碼顯示的亂碼文字。 |
| **文字懸停效果** | 懸停時滑鼠跟隨的漸層與描邊文字。 |
| **向上滑動字母** | 可選標題與說明的滑動字母。 |
| **動畫計數器** | 進入視窗時從起始值動畫到目標值的數字。 |
| **計時器** | 經過時間計時器；開始、暫停、重啟；可設定顯示格式。 |
